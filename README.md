# Sistem Kehadiran Karyawan

Project ini dibuat untuk tugas Praktikum Pemrograman Web dengan menggunakan framework Laravel dan konsep MVC.

## Fitur
- Halaman Home
- Halaman Contact
- Halaman About
- List Barang

## Teknologi
- PHP
- Laravel
- Blade Template

## Cara Menjalankan
1. Clone repository
2. Jalankan:
php artisan serve
3. Buka:
http://localhost:8000
